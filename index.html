Cara Setup Domain

Langkah-langkah konfigurasi domain ke VPS Anda

1

Pembelian & Aktivasi

Kontak kami via Telegram @opendomainmurmer dengan format pesan otomatis. Transfer sesuai harga dan konfirmasi pembayaran. Domain aktif dalam 1-5 menit.

2

Konfigurasi Cloudflare

Daftar Cloudflare gratis → Add Site → Ganti nameserver domain dengan nameserver Cloudflare. Tunggu propagasi 1-24 jam.

3

Pointing ke VPS

Buat A Record di Cloudflare: Name @ atau subdomain, Content IP VPS Anda. Tambahkan CNAME untuk www jika diperlukan.

4

Setup Server & SSL

Konfigurasi web server (Nginx/Apache) untuk listen domain. Install SSL dari Cloudflare Origin Certificates atau gunakan Flexible SSL.
